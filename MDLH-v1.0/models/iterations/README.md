# Iteration Snapshots

Use this folder for dated snapshots of the model lineage archive.

## Snapshot Index

- `2026-04-30/` historical model reports and the first theory repair notes
- `2026-05-01/` model-lineage report and simulation output
- `2026-05-05/` current cleaned model snapshot

## Rules

- keep model folders lowercase;
- keep the snapshot date in the folder path;
- keep filenames date-stamped;
- do not mix raw incoming files with final snapshots;
- keep `source/`, `final/`, `theory/`, and `summary/` separate when used.
