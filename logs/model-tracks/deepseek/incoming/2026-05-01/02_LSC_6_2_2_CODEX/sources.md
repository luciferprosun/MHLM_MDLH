# Source Notes For LSC 6.2.2

Primary or near-primary sources used for the 6.2.2 continuation package:

- BEST PRL abstract: `10.1103/PhysRevLett.128.232501`
- BEST PRC details: `10.1103/PhysRevC.105.065502`
- GALLEX first source paper: `10.1016/0370-2693(94)01586-2`
- GALLEX final source paper: `10.1016/S0370-2693(97)01562-1`
- GALLEX reanalysis: `10.1016/j.physletb.2010.01.030`
- SAGE Cr-51 source measurement: `10.1103/PhysRevC.59.2246`
- SAGE Ar-37 source measurement: `10.1103/PhysRevC.73.045805`
- Gallium line cross sections: `10.1103/PhysRevC.56.3391`
- Updated gallium cross section review: `10.1103/PhysRevC.108.035502`

Extracted working values follow the rounded values in the local dataset and
the published abstracts where available.

The 6.2.2 revision uses the same source base as 6.2.1 but changes the formal
interpretation of the detector tensor:

- isotropic trace is explicit,
- anisotropy is traceless,
- sidereal dependence requires a fixed external frame.
