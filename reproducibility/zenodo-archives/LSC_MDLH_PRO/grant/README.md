# Grant Materials

This directory contains grant-facing framing notes for the MHLM / MDLH archive.

The intended framing is:

- open-source audit infrastructure,
- AI hallucination correlation analysis,
- scientific-reasoning provenance,
- reproducible prompt and response archival.

These notes are included because the public MHLM record is also being prepared
as a grant-discussion artifact, not only as a paper package.
