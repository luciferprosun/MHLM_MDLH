# LSC Theory by LLM

This archive stores public snapshots of how the LSC theory line evolved across
multiple language models.

It is designed for provenance review, not for model worship. The goal is to
show:

- what each model added,
- what each model weakened or challenged,
- how corrections entered the theory line,
- and where coherence increased faster than validation.

## Layout

```text
LSC_theory_by_LLM/
  2026-05-05/
  model_tracks/
```

## Reading Order

1. `model_tracks/README.md`
2. `2026-05-05/INDEX.md`
3. the model folder or track you want to audit
