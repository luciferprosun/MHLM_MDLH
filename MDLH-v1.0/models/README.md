# Model Iterations

This folder is the readable entry point for model-by-model archive snapshots.

## Use

- `experiments/model_reports/` keeps the original model reports.
- `models/iterations/` keeps the cleaned dated snapshots for GitHub.
- Each dated snapshot contains one folder per model or summary track.
- Each model folder uses `source/`, `final/`, and `theory/` where needed.

## Snapshots

- `models/iterations/2026-04-30/` historical model reports and theory updates
- `models/iterations/2026-05-01/` lineage report and simulation summary
- `models/iterations/2026-05-05/` current cleaned model snapshot
