# 2026-05-05 MHLM Iteration Archive Extension

This note documents the 2026-05-05 extension of the public MHLM / MDLH Zenodo
package.

## Why this extension was prepared

The original published package centered on the paper and minimal reproducibility
assets. The repository later accumulated a richer audit trail that is relevant
to the core MHLM claim: how multiple language models can reinforce, challenge,
repair, and narratively stabilize a speculative scientific line before external
validation is complete.

## Added archive layers

- `experiments/`
- `prompt-archive/`
- `llm-simulation-lab/`
- `model-lineage-app/`
- `mhlm/`
- `iterations/LSC_theory_by_LLM/`
- `grant/`

## Main public benefit

The record now exposes the difference between:

- paper-level argument,
- prompt-level provenance,
- model-specific theory evolution,
- hallucination-risk tracking,
- and grant-facing prototype framing.

## Scientific status

This extension does not validate the LSC theory line.

It strengthens the MHLM publication as an auditable research artifact by making
the model-lineage evidence and iteration archive public in a structured form.
