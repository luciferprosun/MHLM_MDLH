# LSC 6.2.2 Public Note

LSC 6.2.2 keeps the existing DOI trail intact while formalizing a repair path for the
LSC detector-response model. The correction makes the isotropic trace explicit,
keeps the directional term traceless, removes the mixed base `1 / E^2` usage, and
anchors sidereal tests in a fixed celestial frame. It is a structure and consistency
update, not a validation claim.
