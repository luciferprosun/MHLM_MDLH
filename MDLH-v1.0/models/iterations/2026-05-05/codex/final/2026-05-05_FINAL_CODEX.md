# Codex Final Vision For MDLH

Date: 2026-05-05

Codex turns the archive into a reproducible validation problem.
Its main contribution is to force LSC-style ideas to survive diagnostics,
leave-one-out checks, and explicit failure modes.

What Codex says LSC must do:

- keep fit diagnostics visible;
- treat exact anchor fits as insufficient;
- separate stability from rhetoric;
- document the repair path after failure.

Codex is the strongest reproducibility layer in this archive. It does not
validate the physics, but it makes the theory line numerically inspectable.
