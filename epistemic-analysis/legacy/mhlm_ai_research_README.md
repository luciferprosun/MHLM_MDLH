# MHLM / AI-Research Wing

Status: working archive

This folder documents the second pole of the project: the use of multiple LLM systems in scientific theory development.

The physics work remains in the LSC data/theory folders. This wing tracks the meta-layer:

- which model produced which analysis;
- what was useful;
- what was unsupported or hallucinated;
- how confidence language changed between models;
- how the theory evolved under multi-model pressure.

## Rules

- AI-generated reports are evidence of the research process, not evidence of physical truth.
- Every model claim needs later source checking.
- Overconfident wording must be preserved in notes when it matters, then corrected in verified summaries.
- Useful synthesis and hallucination should both be recorded.

## Public Framing

When this wing is referenced externally, describe it as an AI-research and reproducibility track, not as proof of any physical claim.
