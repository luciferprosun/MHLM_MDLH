*Purpose:* Reproducible benchmark framework to stress-test Large Language Models against complex scientific datasets. Detects correlated failure modes and non-causal reasoning in LLM outputs.

*Use case:* Detect hallucinated scientific claims in LLM outputs used in research, education, and automated reporting pipelines.

*Origin:* Developed 2023-2026 during anomaly analysis research. Generalized into an AI audit framework.

*Funding track:* Data Security & AI Safety

# LSC 6.0: Unified Curvature-Coupled Neutrino Model

*LSC 6.0* presents a phenomenological framework for testing whether curvature-coupled energy reconstruction can contribute to neutrino-anomaly interpretation without assuming sterile neutrinos as the only possible explanation.

The research line has now been extended to **LSC 6.2.0**, a preprint-stage continuation focused on detector-frame tensor anisotropy. LSC 6.2.0 is not presented as confirmed new physics; it is an arXiv-ready phenomenological ansatz awaiting category endorsement and intended for independent review, numerical fitting, and falsification.

### Latest Preprint: LSC 6.2.0

LSC 6.2.0 extends the public LSC 6.0 framework with an explicit detector-frame rank-2 tensor:

```text
E_rec = E_true [1 + alpha_D D_ij p^i p^j]
```

The draft separates propagation-level true energy from detector-level reconstructed energy and defines validation targets across gallium source experiments, KATRIN, IceCube, and standard three-flavor oscillation limits.

Status:

- Zenodo preprint: https://doi.org/10.5281/zenodo.19878587
- arXiv-ready source package included in [`releases/LSC_6.2.0_preprint`](releases/LSC_6.2.0_preprint/README.md)
- arXiv endorsement pending
- not confirmed new physics

### Testable Prediction

LSC 6.0 proposed a 1-2% sidereal modulation test in BEST detector count rate. LSC 6.2.0 generalizes the test language into detector-frame tensor anisotropy: angular anisotropy, detector dependence, energy-dependent residuals, and sidereal modulation.
