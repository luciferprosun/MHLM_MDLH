# Theory Updates

This folder stores dated notes about changes to the LSC theory line itself.
It is distinct from `model_reports/`, which records what each model said or
changed.

Use this folder for:

- version corrections,
- formal repair notes,
- simulation-driven adjustments,
- and theory-line deltas that are not tied to a single model report.

Key public note:

- `2026-04-30_public_note.md` for a short DOI-side summary.
