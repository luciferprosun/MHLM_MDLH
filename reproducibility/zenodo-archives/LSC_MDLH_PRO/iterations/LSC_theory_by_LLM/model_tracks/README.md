# Model Tracks

This directory groups the long-lived role of each model in the LSC / MHLM
archive.

Each model folder may contain:

- `README.md` - what the model role is,
- `notes.md` - dated notes on contribution and correction,
- `artifacts/` - selected PDFs or exports worth preserving publicly.

These tracks complement the dated snapshots under `../2026-05-05/`.
