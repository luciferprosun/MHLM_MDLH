# LSC 6.2.2

Corrected continuation folder for the LSC 6.2.x line.

## Goal

Fix the formal problems identified in LSC 6.2.0 / 6.2.1:

- make the detector term dimensionally consistent,
- separate isotropic trace from traceless anisotropy,
- define the reference frame for sidereal modulation,
- keep the validation layer reproducible.

## Contents

- `codex/` - reproducible analysis package
- `data/` - canonical gallium source dataset used by the package
- `codex/out/` - PDF, CSV, JSON and simulation digests generated by the build

## Status

This is still a phenomenological framework, not confirmed physics.
The correction is formal and structural, not a claim of validation.
